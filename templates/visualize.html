DATA VISUALIZATION

Data visualization is the representation of data or information in a graph, chart, or other visual format. It communicates relationships of the data with images. Different types of data visualizations present are:
Histogram
Bar plot
Scatter plot
Factor plot etc.

1.Histogram

A Histogram is representation of the distribution of numerical data, where the data are binned and the count for each bin is represented. More generally, a Histogram is an aggregated bar chart, with several possible aggregation functions.




2.Barplot

A bar plot or bar chart is a graph that represents the category of data with rectangular bars with lengths and heights that is proportional to the values which they represent. The bar plots can be plotted horizontally or vertically. A bar chart describes the comparisons between the discrete categories.





3.Scatter plot

A scatter plot is a diagram where each value in the data set is represented by a dot.




4.Correlation Matrix

It is a table which displays the correlation between variables i.e. correlation coefficient. It explains how one or more variables are related to each other. It helps us to predict target variable.
